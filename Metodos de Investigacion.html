Para el proyecto del Colegio "Francisco de Miranda", se empleó el método deductivo como enfoque principal para el análisis, diseño y seguimiento del sistema de control académico. Este método se basa en la aplicación de principios generales para llegar a conclusiones específicas. En primer lugar, se establecieron los principios fundamentales del sistema educativo y los objetivos generales del proyecto. A partir de ahí, se procedió a identificar los requisitos específicos del colegio y las necesidades de los usuarios, como padres, profesores y personal administrativo. Con base en estos principios generales y requisitos específicos, se desarrolló un marco conceptual que sirvió como base para el diseño del sistema de seguimiento académico. Se formularon hipótesis sobre cómo el sistema podría abordar eficazmente las necesidades identificadas y mejorar el proceso educativo en el colegio. Luego, se procedió a diseñar el sistema utilizando estas hipótesis como guía. Se definieron las estructuras de datos, los algoritmos y las funcionalidades necesarias para cumplir con los objetivos del proyecto. Se utilizaron técnicas de modelado y prototipado para visualizar y validar el diseño del sistema antes de su implementación. Una vez que el diseño del sistema estuvo completo, se llevó a cabo la implementación y puesta en marcha del mismo en el Colegio "Francisco de Miranda". Durante esta etapa, se realizaron pruebas y ajustes para garantizar que el sistema cumpliera con los requisitos y expectativas del colegio y sus usuarios.
